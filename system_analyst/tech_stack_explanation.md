# Обоснование выбора технологического стека

## Backend

### Python 3.12+
Современная версия Python с поддержкой последних возможностей языка, включая улучшенную типизацию и производительность.

### FastAPI
Асинхронный веб-фреймворк, обеспечивающий высокую производительность и автоматическую генерацию OpenAPI-документации. Идеально подходит для создания API с поддержкой асинхронных операций.

### PostgreSQL
Надежная реляционная СУБД с поддержкой JSONB для гибкого хранения полуструктурированных данных. Поддержка расширений, таких как pgvector, позволяет использовать PostgreSQL и для векторных данных в будущем.

### SQLAlchemy 2.0+
Современный ORM с поддержкой асинхронных операций и мощными возможностями для работы с базами данных. Сочетание удобства работы с объектами и гибкости SQL.

### Pydantic
Библиотека для валидации данных и сериализации, используемая в FastAPI. Обеспечивает строгую типизацию и валидацию входящих и исходящих данных.

### Valkey
Сообщество-поддерживаемый форк Redis, сохраняющий полную совместимость с API Redis. Обеспечивает надежное решение для кэширования и хранения сессий.

### RabbitMQ
Надежный брокер сообщений с поддержкой различных паттернов обмена сообщениями. Подходит для построения асинхронной архитектуры и обработки фоновых задач.

### Библиотеки аутентификации
- `python-jose` для работы с JWT-токенами
- `httpx` для взаимодействия с OAuth-провайдерами

## Frontend

### React
Отраслевой стандарт для создания сложных интерактивных интерфейсов. Богатая экосистема и широкое сообщество разработчиков.

### Vite
Современный инструмент сборки, обеспечивающий быструю загрузку в режиме разработки и эффективную сборку для продакшена.

### Tailwind CSS
Utility-first фреймворк, позволяющий быстро создавать адаптивные и кастомные интерфейсы без написания дополнительного CSS.

## DevOps и инфраструктура

### Docker
Контейнеризация обеспечивает согласованность сред выполнения на всех этапах разработки и деплоя.

### HashiCorp Vault
Центральное решение для управления секретами, учетными данными и шифрованием.

### Selectel и Yandex Cloud
Выбор российских облачных провайдеров обеспечивает соответствие требованиям локализации данных и снижает задержки для конечных пользователей.

### Poetry
Современный менеджер зависимостей для Python, обеспечивающий воспроизводимость сборки и управление виртуальными окружениями.

### Pytest
Мощная библиотека для тестирования Python-приложений с широкими возможностями для создания и запуска тестов.

## Уведомления

### Email
Традиционный канал для формальных и несрочных уведомлений.

### Telegram (python-telegram-bot)
Современный канал для мгновенных операционных уведомлений, обеспечивающий высокую вовлеченность пользователей.
