# Feasibility Study

> **Meta:** v1.0.0 | 22-02-2026

## Назначение

Навык для проведения технико-экономического обоснования проекта, включающий анализ осуществимости проекта с технической, экономической, временной и ресурсной точек зрения. Помогает определить, стоит ли продолжать проект, и какие риски могут возникнуть.

## Когда использовать

Используйте этот навык:
- На этапе инициации проекта
- При необходимости оценить осуществимость проекта
- Когда нужно принять решение о продолжении или прекращении проекта
- При подготовке бизнес-обоснования для руководства

## Функции

### Technical Feasibility
Оценка технической осуществимости:
- Анализ требований к технической инфраструктуре
- Оценка доступных технологий и их соответствия требованиям
- Анализ сложности реализации
- Оценка рисков, связанных с технологиями
- Проверка соответствия техническим стандартам

### Financial Analysis
Финансовый анализ проекта:
- Оценка затрат на реализацию
- Прогноз ожидаемой выгоды
- Расчет ROI, NPV, IRR
- Оценка срока окупаемости
- Анализ чувствительности

### Resource Requirements
Анализ ресурсных требований:
- Оценка необходимых человеческих ресурсов
- Оценка материальных ресурсов
- Оценка времени на реализацию
- Анализ доступности ресурсов
- Проверка соответствия квалификации команды требованиям

### Risk Assessment
Оценка рисков проекта:
- Идентификация потенциальных рисков
- Оценка вероятности и влияния рисков
- Оценка возможных последствий
- Проверка доступности мер по снижению рисков
- Анализ угроз и возможностей

## Интеграция с System Analyst

- Получение технических требований от System Analyst
- Анализ архитектурных решений на предмет осуществимости
- Учет рекомендаций по технологиям от System Analyst
- Оценка рисков, связанных с техническими решениями
- Валидация ресурсных требований с технической точки зрения

## Примеры использования

### Пример 1: Оценка осуществимости разработки мобильного приложения
Техническая осуществимость:
- Поддержка необходимых функций в iOS и Android: Да
- Требуемые API доступны: Да
- Требуемая производительность достижима: Да
- Оценка сложности интеграции с существующими системами: Средняя

Финансовый анализ:
- Затраты на разработку: 800,000 руб.
- Ожидаемая экономия: 1,200,000 руб./год
- ROI: 150% за первый год
- Срок окупаемости: 8 месяцев

Ресурсные требования:
- Необходимо 6 разработчиков на 4 месяца
- Требуется 1 UI/UX дизайнер на 2 месяца
- Необходимо 2 тестировщика на 2 месяца
- Доступность ресурсов: Достаточна

### Пример 2: Оценка осуществимости внедрения ERP-системы
Техническая осуществимость:
- Совместимость с существующей инфраструктурой: Да
- Требуемая вычислительная мощность доступна: Да
- Возможность интеграции с текущими системами: Да
- Необходимость миграции данных: Сложная, но выполнима

Финансовый анализ:
- Затраты на лицензии: 1,500,000 руб.
- Затраты на внедрение: 2,000,000 руб.
- Ожидаемая экономия: 1,000,000 руб./год
- ROI: 28% за 3 года
- Срок окупаемости: 3.5 года
