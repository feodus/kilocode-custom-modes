# Project Budget Planning

> **Meta:** v1.0.0 | 22-02-2026

## Назначение

Навык для детального бюджетирования проекта, включая оценку капитальных и операционных затрат, планирование денежных потоков и формирование бюджетных периодов. Обеспечивает финансовую прозрачность и контроль над затратами проекта.

## Когда использовать

Используйте этот навык:
- На этапе инициации и планирования проекта
- При необходимости формирования финансового обоснования
- Когда нужно определить бюджет проекта с разбивкой по статьям расходов
- При подготовке отчетности для спонсоров и заинтересованных сторон

## Функции

### Cost Estimation
Оценка стоимости проекта по различным категориям:
- Трудозатраты (зарплаты, контракторы)
- Оборудование и программное обеспечение
- Инфраструктура и облачные услуги
- Обучение и сопровождение
- Прочие операционные расходы

### CAPEX/OPEX Planning
Планирование капитальных и операционных затрат:
- CAPEX: единовременные инвестиции в активы
- OPEX: текущие операционные расходы
- Оптимизация структуры затрат
- Сравнение моделей финансирования

### Budget Allocation
Распределение бюджета по периодам и компонентам:
- Бюджет по фазам проекта
- Бюджет по командам и функциям
- Резерв на непредвиденные расходы
- Учет сезонности и временных факторов

### Financial Tracking
Отслеживание и контроль бюджета:
- Мониторинг отклонений
- Прогнозирование оставшихся затрат
- Управление изменениями бюджета
- Финансовая отчетность

## Интеграция с System Analyst

- Получение данных о затратах на разработку от System Analyst
- Учет архитектурных решений в бюджете
- Анализ влияния технических решений на стоимость
- Валидация оценок трудозатрат
- Учет рисков и их финансового влияния

## Примеры использования

### Пример 1: Бюджетирование проекта разработки веб-приложения
CAPEX:
- Разработка: 500,000 руб. (10 чел*мес * 50,000 руб/мес)
- Тестирование: 150,000 руб. (3 чел*мес * 50,000 руб/мес)
- Инфраструктура: 200,000 руб. (оборудование, лицензии)

OPEX:
- Поддержка: 120,000 руб./мес (2 чел*мес * 60,000 руб/мес)
- Инфраструктура: 30,000 руб./мес (облако, лицензии)
- Обучение: 50,000 руб. (однократно)

### Пример 2: Бюджетирование проекта внедрения ERP-системы
CAPEX:
- Лицензии: 2,000,000 руб.
- Консалтинг: 1,500,000 руб.
- Интеграция: 800,000 руб.
- Оборудование: 500,000 руб.

OPEX:
- Поддержка: 200,000 руб./мес
- Обновления: 100,000 руб./год
- Обучение: 150,000 руб./год
