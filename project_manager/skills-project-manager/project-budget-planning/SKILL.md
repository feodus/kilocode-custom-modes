# Project Budget Planning

> **Meta:** v1.0.0 | 22-02-2026

## Purpose

Skill for detailed project budgeting, including estimation of capital and operational expenses, cash flow planning, and forming budget periods. Provides financial transparency and cost control over the project.

## When to Use

Use this skill:
- At project initiation and planning stage
- When forming financial justification is needed
- When determining project budget with breakdown by expense categories
- When preparing reporting for sponsors and stakeholders

## Functions

### Cost Estimation
Project cost estimation by various categories:
- Labor costs (salaries, contractors)
- Equipment and software
- Infrastructure and cloud services
- Training and support
- Other operational expenses

### CAPEX/OPEX Planning
Planning capital and operational expenses:
- CAPEX: one-time investments in assets
- OPEX: ongoing operational expenses
- Cost structure optimization
- Financing model comparison

### Budget Allocation
Budget allocation by periods and components:
- Budget by project phases
- Budget by teams and functions
- Contingency reserve
- Accounting for seasonality and temporal factors

### Financial Tracking
Budget tracking and control:
- Monitoring deviations
- Forecasting remaining costs
- Budget change management
- Financial reporting

## Integration with System Analyst

- Receiving development cost data from System Analyst
- Accounting for architectural decisions in budget
- Analyzing impact of technical decisions on cost
- Validating effort estimates
- Accounting for risks and their financial impact

## Usage Examples

### Example 1: Budgeting for Web Application Development Project
CAPEX:
- Development: 500,000 rubles (10 person-months * 50,000 rubles/month)
- Testing: 150,000 rubles (3 person-months * 50,000 rubles/month)
- Infrastructure: 200,000 rubles (equipment, licenses)

OPEX:
- Support: 120,000 rubles/month (2 person-months * 60,000 rubles/month)
- Infrastructure: 30,000 rubles/month (cloud, licenses)
- Training: 50,000 rubles (one-time)

### Example 2: Budgeting for ERP System Implementation Project
CAPEX:
- Licenses: 2,000,000 rubles
- Consulting: 1,500,000 rubles
- Integration: 800,000 rubles
- Equipment: 500,000 rubles

OPEX:
- Support: 200,000 rubles/month
- Updates: 100,000 rubles/year
- Training: 150,000 rubles/year
