# Targeted Advertising

> **Meta:** v1.1.0 | 08-03-2026

## Purpose

Skill for setting up and managing targeted advertising in social networks and messengers. Includes working with audiences, creating creatives, setting up targeting, and optimizing advertising campaigns in VK, Telegram, and myTarget.

## When to Use

Use this skill:
- When needing to launch targeted advertising
- For setting up retargeting to existing customers
- When working with look-alike audiences
- For optimizing acquisition cost
- When launching advertising campaigns in new channels

## Functions

### Audience Setup
- Creating basic audiences by demographics
- Setting up behavioral audiences
- Working with interests and lifestyles
- Building look-alike audiences
- Using retargeting
- Uploading and segmenting customer bases

### Targeting Configuration
- Geographic targeting
- Time targeting
- Frequency settings
- Audience exclusions
- Cross-platform targeting

### Creative Development
- Creating advertising banners
- Writing advertising texts
- Adapting creatives for formats
- A/B testing visuals
- Developing video creatives

### Campaign Optimization
- Analyzing placement effectiveness
- Optimizing bids
- Scaling successful campaigns
- Working with fraud
- Optimizing conversions

## Integration with Universal Coding Agent

- Delegate technical pixel integration
- Request creation of landing pages
- Delegate UTM tag setup
- Request technical call tracking implementation

## Usage Examples

### Example 1: VK campaign setup
**Business:** Online clothing store

```
Campaign: Clothing_Spring2026
├── Audience: Women 25-45, Moscow
├── Interests: Fashion, Shopping, Clothing
├── Format: Carousel
└── Creatives: 5 banners with different products
```

### Example 2: Audience segmentation
| Segment | Description | Size | Bid |
|---------|----------|--------|--------|
| Loyal | Purchases > 3 | 5,000 | 50 rub |
| New | In cart, didn't buy | 15,000 | 35 rub |
| Similar | Look-alike of loyal | 50,000 | 25 rub |
| Cold | Interests | 200,000 | 15 rub |
