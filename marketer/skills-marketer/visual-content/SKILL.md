# Visual Content

> **Meta:** v1.1.0 | 08-03-2026

## Purpose

Skill for creating and coordinating visual content. Includes developing briefs for designers, creating visual concepts, branding advertising materials.

## When to Use

Use this skill:
- When creating advertising banners
- For developing presentations
- When creating infographics
- For preparing visuals for social networks
- When launching advertising campaigns

## Functions

### Creative Brief Development
- Formulating requirements
- Describing target audience
- Defining key messages
- Technical requirements
- Brandbook requirements

### Visual Concept
- Developing concept
- Selecting references
- Color solution
- Typography
- Style

### Brand Compliance
- Logo usage
- Fixed colors
- Fonts and styles
- Unified visual language
- Compliance verification

### Content Types
- Static banners
- Animated banners
- Video creatives
- Infographics
- Presentations

## Usage Examples

### Example 1: Banner brief
```
Brief: Banner for Yandex Direct

Product: Python online course
Size: 300x250
Target audience: Beginner programmers 20-35 years old

Visual:
- Background: Dark with gradient (blue-purple)
- Center: Laptop with code
- Text: "Master Python in 3 months"
- Button: "Start learning"

Requirements:
- Logo in top right corner
- No more than 3 colors
- Clear readable text
- Format: PNG, JPG
```

### Example 2: Quality checklist
- [ ] Follows brandbook
- [ ] Correct sizes
- [ ] Readable text
- [ ] Call to action (CTA)
- [ ] Logo in place
- [ ] Tested on different screens

### Example 3: References for inspiration
| Category | Style | Application |
|-----------|-------|------------|
| Tech | Minimalism, blue accents | IT products |
| Fashion | Bright colors, large photos | Clothing, accessories |
| Finance | Serious, dark tones | Banks, investments |
| Food | Natural colors, appetizing photos | Food, delivery |

### Example 4: Infographic
```
Brief: Infographic "How to choose a course"

Structure:
1. Header with icon
2. 5 steps with icons
3. Comparison table
4. Call to action

Style:
- Minimalism
- 3 main brand colors
- Icons in unified style
- Infographic SVG or AI
```
