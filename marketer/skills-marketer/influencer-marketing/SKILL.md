# Influencer Marketing

> **Meta:** v1.1.0 | 08-03-2026

## Purpose

Skill for working with influencers and bloggers. Includes search, analysis, negotiations, planning, and coordinating placements.

## When to Use

Use this skill:
- When launching an advertising campaign with bloggers
- For increasing brand awareness
- When working with opinion leaders
- For managing influencer programs
- When evaluating placement effectiveness

## Functions

### Influencer Research
- Profile analysis
- Audience assessment
- Engagement checking
- Content analysis
- Fraud checking

### Negotiation
- Defining terms
- Price discussion
- Placement format
- Exclusivity
- Commercial terms

### Campaign Planning
- Content plan
- Script coordination
- Timing
- Integration with other channels
- Expected results

### Coordination
- Communication with influencer
- Execution control
- Material verification
- Payment and reporting
- Post-campaign analysis

## Influencer Types

| Type | Subscribers | Avg ER | Cost |

## Usage Examples

### Example 1: Finding influencers
```
Niche: Education / Self-development

Criteria:
- Audience: 25-45 years old
- Topics: Self-development, business, education
- ER: > 3%
- Geography: Russia

Found influencers:
| Blogger | Subscribers | ER | Cost | Reach |
|--------|------------|-----|-----------|-------|
| @coach1 | 250K | 4.2% | 80K ₽ | 50K |
| @bizlady | 180K | 3.8% | 60K ₽ | 35K |
| @studypro | 120K | 5.1% | 45K ₽ | 30K |
```

### Example 2: Placement structure
```
Format: Native integrated post

Channel: YouTube (video review)
Duration: 10-15 minutes
Cost: 150,000 ₽

Script:
1. Introduction (1 min) - meet the blogger
2. Problem (2 min) - audience pain point
3. Solution (5 min) - product presentation
4. Experience (3 min) - personal usage experience
5. CTA (1 min) - link click

Materials from us:
- Product for review
- Key points
- FAQ for blogger
- Special promo code
```
