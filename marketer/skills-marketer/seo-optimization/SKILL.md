# SEO Optimization

> **Meta:** v1.1.0 | 08-03-2026

## Purpose

Skill for conducting search engine optimization of websites and landing pages. Includes working with keywords, meta tags, technical SEO, and building link mass.

## When to Use

Use this skill:
- When needing to improve website positions in search results
- For conducting SEO audit of existing website
- When creating a new website or landing page
- For optimizing content for target keywords
- When setting up technical SEO

## Functions

### Keyword Research
- Analyzing and selecting relevant keywords
- Evaluating query competitiveness
- Determining frequency and intents
- Clustering queries by topics

### On-Page SEO
- Optimizing meta tags (title, description)
- Setting up H1-H6 headings
- Content optimization
- Working with internal links
- Image optimization (alt, title)

### Technical SEO
- Setting up sitemap.xml
- Optimizing robots.txt
- Improving loading speed
- Setting up structured data (Schema.org)
- Mobile optimization
- Working with Core Web Vitals

### Off-Page SEO
- Building link mass
- Analyzing competitor backlinks
- Working with business directories
- Setting up local SEO

## Integration with Universal Coding Agent

For technical implementation of SEO recommendations:
- Delegate technical SEO setup to Universal Coding Agent
- Request loading speed optimization
- Delegate structured data creation
