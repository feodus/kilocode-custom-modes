# Email Marketing

> **Meta:** v1.1.0 | 08-03-2026

## Purpose

Skill for creating and managing email marketing campaigns. Includes email development, subscriber base segmentation, automation setup, and A/B testing.

## When to Use

Use this skill:
- When needing to create email newsletters
- For setting up automatic email chains
- When segmenting subscriber base
- For increasing conversion through email channel
- When optimizing existing newsletters

## Functions

### Email Campaign Management
- Planning regular newsletters
- Creating various email types
- Setting up sending
- Analyzing results

### List Segmentation
- Segmentation by behavior
- Demographic segmentation
- RFM analysis
- Creating interest groups

### Marketing Automation
- Setting up trigger emails
- Creating welcome chains
- Setting up abandoned carts
- Lifecycle marketing

### A/B Testing
- Testing email subject
- Testing sending time
- Testing content
- Optimization based on results

## Integration with Universal Coding Agent

- Delegate technical email service integration
- Request API integration setup
- Delegate creation of landing pages for email collection

## Usage Examples

### Example 1: Welcome chain structure
**Business:** Online courses

**Email chain:**

| Email | Day | Subject | Content |
|--------|------|------|---------|
| 1 | 0 | Welcome! | Introduction, what to expect |
| 2 | 1 | How to start learning | Step-by-step instructions |
| 3 | 3 | Success story | Student review |
| 4 | 7 | First result | Motivation |
| 5 | 14 | Special offer | Bonus for payment |

### Example 2: Subscriber base segmentation
| Segment | Criteria | Email |
|---------|----------|--------|
| New subscribers | < 30 days | Welcome series |
| Active | Opened >50% emails in 30 days | Exclusive content |
| Inactive | Didn't open >60 days | Reactivation |
| Buyers | Made a purchase | Upsell |
