# Affiliate Marketing

> **Meta:** v1.1.0 | 08-03-2026

## Purpose

Skill for launching and managing affiliate programs. Includes working with partners and webmasters, setting up tracking, optimizing commissions, and scaling the affiliate network.

## When to Use

Use this skill:
- When launching an affiliate program
- For scaling customer acquisition
- When working with affiliate networks
- For managing partners
- When optimizing affiliate marketing

## Commission Structure

### Typical Models

| Model | Description | Example |
|--------|----------|--------|
| CPS (Cost per Sale) | % of sale | 10% of order |
| CPA (Cost per Action) | Per action | 500 ₽ per registration |
| RevShare | % of revenue | 30% of lifetime revenue |
| Hybrid | Combination | Base rate + bonus |

### Partner Tiers

| Tier | Sales/month | Commission | Bonus |
|---------|--------------|----------|-------|
| Bronze | 0-50 | 10% | - |
| Silver | 51-200 | 12% | +5% on turnover |
| Gold | 201-500 | 15% | Personal manager |
| Platinum | 500+ | 20% | Exclusive terms |

## Usage Examples

### Example 1: Launching an affiliate program
```
Program: "Refer a Friend"

Terms:
- Referral receives: 500 ₽ discount
- Partner receives: 300 ₽ per sale
- Cookie: 30 days

Promo materials:
- Referral link
- Banners in 3 sizes
- Promo codes
- Email templates
```
