# VSCode Codicons Reference

Based on the Orchestrator mode configuration, the icons for Kilocode custom modes appear to come from the VSCode Codicons set. The Orchestrator mode specifically uses `codicon-run-all` as its icon.

## About VSCode Codicons

VSCode Codicons are the official icon font used by Visual Studio Code. They provide a consistent set of icons that integrate well with the VSCode interface. The icons are referenced by their name prefixed with `codicon-` in the YAML configuration files.

The codicon icon set includes over 200 different icons covering development, collaboration, and management activities, making them suitable for representing different specialized modes in Kilocode.
